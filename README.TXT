In order to use the AudioExtractor, a few packages must be installed

1: yt-dlp - Used to download the video from Youtube
MAC/Linux:
Homebrew
Follow instructions and install homebrew at https://brew.sh/
Then type 

brew install yt-dlp

into a terminal
After it installs, type 

brew install ffmpeg


Make sure to have python installed, then type in

pip install moviepy


Windows:
Run the following commands

pip install yt-dlp
pip install moviepy
pip install ffmpeg



Enter the URLs for the youtube videos you want your song to come from in the songs.txt file
Specify the locations of where you want your songs to end up in CONF.txt, located in the assets folder
EX: /Users/[username]/Desktop/songs

Run and enjoy
